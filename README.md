# 📊 Travel & Passport Analytics Dashboard

## 📌 Project Overview

This project is an end-to-end data analytics solution that simulates real-world travel and passport data. It covers the complete data workflow — from data generation and storage to analysis and visualization — to extract meaningful, decision-oriented insights.

The goal of this project is to demonstrate practical data analytics skills using industry-relevant tools and techniques.

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy, Faker) – Data generation & preprocessing
* **MySQL** – Data storage & querying
* **SQL** – Data analysis (aggregation, joins, trend analysis)
* **Power BI** – Interactive dashboard & visualization

---

## 📂 Dataset

* Generated **100,000+ synthetic travel records**
* Simulates real-world features such as:

  * Passenger demographics (age, gender)
  * Travel country & airport
  * Travel purpose (business, tourism, etc.)
  * Passport & travel dates

---

## ⚙️ Project Workflow

### 1️⃣ Data Generation (Python)

* Created large-scale synthetic dataset using Faker
* Cleaned and structured data using Pandas
* Prepared dataset for database insertion

### 2️⃣ Data Storage (MySQL)

* Designed relational schema
* Inserted data in batches for efficiency
* Managed structured storage for querying

### 3️⃣ Data Analysis (SQL)

* Performed:

  * Aggregations (COUNT, AVG, GROUP BY)
  * Trend analysis (monthly/yearly)
  * Country-wise and demographic insights
* Extracted key patterns in travel behavior

### 4️⃣ Data Visualization (Power BI)

* Built an interactive dashboard including:

  * KPI Cards (Total Travelers, Trends)
  * Monthly travel trends
  * Country-wise distribution
  * Demographic analysis (Age, Gender)
  * Travel purpose insights

---

## 📈 Key Insights

* High concentration of travelers across select countries
* Strong inbound travel dominance
* Noticeable patterns in travel purpose and demographics
* Consistent monthly travel trends with peak periods
